Computers & Turtles

Computers, Let's start with the normal computers, they're basically turtles but worse since they can't move, break blocks, or fight. They can however execute any javascript code.

Monitor Computers, Unlike many mods out there we don't have seperate monitors but our monitors are also computers, bascially monitor computers are computers but with an editable map infront, they can also execute javascript code just like normal computers.

Turtles, Basically computers with wheels, they can execute javascript code but can also move, break blocks and pick up items making them useful for automated tasks such as mining or tree harvesting. They can also place blocks, so if you want an automated house build you can do that too.

Sword Turtles, A different type of turtle that has a sword instead of a pickaxe, sacrifising the ability to break blocks with the ability to hit mobs and players. The great part is that unlike some other mods turtle kills count as player kills so you stilll get the correct mob loot.
