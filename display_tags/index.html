Display Tags

Not sure why Minecraft: Java Edition doesn't already have display names, like everything else does. (even Reddit lmao)

You can view and modify your display tag using /display. It cannot be used while you have a front set.
If your account is a plural system, you can view and modify your members' display tags using /member <name> display with the same syntax as /display.

A display tag is comprised of 4 parts:

  • a display name, which can be modified with /display name [name]. A display name CAN contain emojis.
  • pronouns, which can be modified with /display pronouns [pronouns].
  • a display color, which can be modified with /display color [hex code]. The hex code does not contain the #.
  • the server operator badge, which exists only for server operators, duh.

You can unset any of the modifiable properties by passing in nothing or you can reset all properties of a display tag by doing /display reset.
You can also view what your display tag looks like with just /display.
