You know how Minecraft servers sometimes have corresponding Discord servers? Yeah...

When enabled, the plugin will also act as a bot for the Discord server. The bot will synchronize the messages in a Discord channel with the Minecraft server's chat.

The server may also decide to enforce a whitelist managed by the Discord server, in which case, you will need to link your Discord account with your Minecraft username.
In order to do so, join both the Discord server and Minecraft server and use the Discord command /whitelist <link code> with the link code provided by the Minecraft server.

You may link multiple Minecraft accounts to one Discord account but not vice versa.
You may unlink a Minecraft account from a Discord account by either using the Discord command /unlink <Minecraft username> or the Minecraft command /unlink.
You may view a list of all linked accounts using the Discord command /links.

If the server uses a Discord-managed whitelist, all messages sent from the Discord channel synchronized to the Minecraft server's chat, will be sent as that Discord account's primary Minecraft account.
In order to change which Minecraft account is your primary account, use the Discord command /setprimary <Minecraft username>.

You can also use /sync to update the data the server knows about you, since it's not omniscient.
