Is your mind a single being? If so then get out of this page, for this is only useful for players with plural systems!
(For the record, I am not a cop, I cannot prevent you from reading this page if you are interested in its material, nor do I have any intention to)

In order to use this feature, you need to mark your account as a plural system using /system enable.
You can reverse the decision by using /system disable.
You can then also name your system using /system name [name] and set the system's tag using /system tag [tag].
In order to unset either of these two properties, use /system name and /system tag respectively.

A plural system is no good without any members, so don't forget to create some with /member <name> create!
You can view a list of all members of your plural system by using
/member list.

You can delete already created members with /member <name> delete, but if you wish to rename the member instead, then you can use /member <name> rename <new name>.

Do you want to modify the member's display tags? Well you can also do that with /member <name> display with the same syntax as the /display command.

In order to actually use the system for its intended purpose, you need to set some proxy tags on your members.
Proxy tags are what you enclose your message in to tell the plugin which member you want to proxy as.
Common proxy tags are [square brackets], {curly brackets} or A:letter prefixes.
And yes, I stole those examples from PluralKit, don't blame me.

In order to set the proxy tags of a member, you need to use /member <name> proxy [tags].
The tags provided need to have text somewhere, which represents the message you are wrapping in the proxy tags.
You can leave out the tags parameter to clear the proxy tags of a member.

You can also set a front for your system, which will automatically proxy all your messages which don't trigger another proxy.
To do so, type in /front <name>. You can also do just /front to clear your front.
Mention that your front gets cleared when you leave.

Also, you cannot use a proxy while using the Discord channel.
And any unproxied messages don't contain any evidence of you being a plural system.

You can optionally set a skin for a member of your plural system.
To do that, you do /member <name> skin [mineskin id].
The command requires a Mineskin ID or URL.
You can remove a member's skin by just doing /member <name> skin.

A member's skin will replace your own while they are set as your front. It will also appear in the linked Discord channel when sending messages, but not when obtaining advancements, joining, quitting or dying.
