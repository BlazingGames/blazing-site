Minor Features

Features that didn't deserve their own pages. Featuring:

  • GoodChat - Typing color codes to stylize your chat messages
  • Discord support - communicate with the server using a Discord channel
  • /killme - suicide. A slow suicide. This made more sense when keepInventory was on the server.
  • /playtime - check how much time you wasted played on this server.
  • /display - change your pronouns, display name and nameplate color.
  • Stylized Custom Names - Use color codes in the anvil to stylize your item's name.
