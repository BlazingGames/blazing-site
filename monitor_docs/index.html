BlazingSite
The Monitor Object Documentation
This page only applies to monitors.
Drawing Functions
setPixel
Set the rgb value of a pixel at the specified xy, true for success, false for errors
setPixel: (x: number, y: number, rgb: number) => boolean;
getPixel
Get the rgb value of a pixel at the specified xy, or -1 for errors
getPixel: (x: number, y: number) => number;
fillRect
Fill an area with a specified rgb color
fillRect: (x: number, y: number, width: number,
		   height: number, rgb: number) => boolean;
Interaction Functions
setClickHandler
Sets the handler for detecting clicks
setClickHandler: (handler: (x: number, y: number) => void)
		         => void;
