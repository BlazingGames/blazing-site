The Turtle Object Documentation
This page only applies to turtles and sword turtles.
These functions are available in the turtle object.
General Functions
isReady
Checks if the turtle is ready for the next action
isReady: () => boolean;
waitTime
Returns the time to wait until the next action can be performed
waitTime: () => number;
Movement Functions
forward
Move the turtle forward
forward: () => boolean;
backward
Move the turtle backward
backward: () => boolean;
up
Moves the turtle up
up: () => boolean;
down
Moves the turtle down
down: () => boolean;
left
Makes the turtle rotate left
left: () => boolean;
right
Makes the turtle rotate right
right: () => boolean;
Block Manipulation Functions
destroy
Destroys the block in front of the turtle
destroy: () => boolean;
destroyAbove
Destroys the block above the turtle
destroyAbove: () => boolean;
destroyBelow
Destroys the block below the turtle
destroyBelow: () => boolean;
place
Place a block in front of the turtle
place: (slot: number) => boolean;
placeAbove
Place a block above the turtle
placeAbove: (slot: number) => boolean;
placeBelow
Place a block below the turtle
placeBelow: (slot: number) => boolean;
look
Gets the block type in front of the turtle
look: () => string;
lookUp
Gets the block type above the turtle
lookUp: () => string;
lookDown
Gets the block type below the turtle
lookDown: () => string;
Attacking Functions
attack
Attacks in the direction the turtle is facing
attack: () => boolean;
Inventory Manipulation Functions
dropAll
Drops the entire turtle inventory
dropAll: () => boolean;
getInventory
Gets the inventory contents of the slot
getInventory: (slot: number) => {
	"type": string,
	"count": string,
	"nbt": string
};
take
Take an item from the container in front of the turtle
take: (slot: number, amount: number) => boolean;
takeBelow
Take an item from the container below the turtle
takeBelow: (slot: number, amount: number) => boolean;
takeAbove
Take an item from the container above the turtle
takeAbove: (slot: number, amount: number) => boolean;
put
Put an item in the container in front of the turtle
put: (slot: number, amount: number) => boolean;
putBelow
Put an item in the container below the turtle
putBelow: (slot: number, amount: number) => boolean;
putAbove
Put an item in the container above the turtle
putAbove: (slot: number, amount: number) => boolean;
getContainer
Gets the contents of the slot of the container in front of the turtle
getContainer: (slot: number) => {
	"type": string,
	"count": string,
	"nbt": string
};
swapContainerSlots
Swaps two slots in the container in front of the turtle
swapContainerSlots: (slotA: number, slotB: number) => boolean;
