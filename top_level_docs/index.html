BlazingSite
The Top Level Documentation
These functions are available at the top level. Deal with it.
Console Functions
startup
Boots the computer, meant to be used in the console
startup: () => void;
shutdown
Shuts down the computer and stops everything
shutdown: () => void;
changePassword
Changes the password of the computer, will be hashed and salted
changePassword: (password: string) => void;
Miscellaneous Functions
sleep
Pauses the execution for the specified amount of time
sleep: (time: number) => boolean;
